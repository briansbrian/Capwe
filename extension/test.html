Testing Instructions

  1. Load the Capwe extension in Chrome (chrome://extensions/)

  • External Links: Yellow border with destination info
  • Forms: Blue border with security info (HTTP forms should show warnings)
  • Hidden Elements: Gray border tooltip when detected
  • AI Analysis: Additional insights when AI is enabled
